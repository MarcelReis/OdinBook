## Requirements

 - lerna - https://lerna.js.org/
 - firebase cli - https://firebase.google.com/docs/cli
 - yarn 1.x - https://yarnpkg.com/
 - node 14.x - https://nodejs.org/
 - nvm - https://github.com/nvm-sh/nvm

## Scripts

`yarn run start`

`yarn run format`

`yarn run lint`

`yarn run test`

`yarn run build`
